# NS Browser Extension

Browser extension to fix the CSS of [NS Actuele Vertrektijden](https://www.ns.nl/reisinformatie/externe-schermen).

## Installation

1. Clone this Git repository or download a ZIP archive.
2. Follow the instructions below for your browser.

### Chromium / Chrome

1. Go to the Extensions page by entering `chrome://extensions` in a new tab.

   - Alternatively, click the Extensions menu puzzle button and select **Manage Extensions** at the bottom of the menu.
   - Or, click the menu, hover over **More Tools**, then select **Extensions**.

2. Enable Developer Mode by clicking the toggle switch next to **Developer mode**.

3. Click the **Load unpacked** button and select the extension directory.

![Extensions page](https://developer.chrome.com/static/docs/extensions/get-started/tutorial/hello-world/image/extensions-page-e0d64d89a6acf_1920.png)

### Firefox

1. Go to the Debugging page by entering `about:debugging` in a new tag.

2. Click **This Firefox** in the menu.

3. Click the **Load Temporary Add-On** button and select the extension manifest (`manifest.json`).
